jsp페이지의 스크립트 요소

선언문: 전역변수 선언 및 메소드 선언  -- <%!문장 %>
<%!
   String= "hi";	//멤버변수 선언

	return id;}

스크립트릿: 프로그래밍 코드 기술 -- <%문장%>
프로그래밍의 로직을 기술/ 일반적으로 가장많이 쓰이는 스크립트요소/ 
여기서 선언된 변수는 지역변수(변수는 큰의미없지만 메소드는 선언못함)


표현식: 화면에 추력할 내용 기술 -- <%=문장%>
jsp페이지에서 웹브라우저에 출력할 부분표현
스크립트릿내에서 출력할부분은 println()메소드 사용(표현식에서는 println()사용못함)

jsp페이지의 디렉티브
jsp페이지의 구성요소중 하나. 웹브라우저가 요청한 jsp페이지가 실행될때 필요한~~

page 디렉티브 -- <%@page%>
jsp페이지에 대한 정보는 page디렉티브의 속성들을 사용해서 정의
속성:info/ language/ contentType: 문서의 타입을 지정/extends/import : 다른 패키지의 클래스 가져와 사용/session/buffer/atuoFlush/isThreadSafe/errorPage/pageEncoding


include 디렉티브 --  <%@include%>
여러jsp페이지에서 공통적으로 사용되는 내용이있을때 이러한 내용을 별도의 파일로 저장해두었다가
필요한  jsp페이지내에 삽입기능
<%@ include file ="포한될 파일의 url"%>

include되는 jsp페이지의 소스내용을 그대로 포함해서 컴파일 / 복사&붙어넣기방식으로 하나의파일로변환되고 컴파일 / 주로 조각코드를 삽입할때 사용
